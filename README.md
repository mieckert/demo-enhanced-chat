# Demonstration Code for Salesforce Enhanced Chat

This repository contains example code for [Salesforce Enhanced Chat Web](https://developer.salesforce.com/docs/service/messaging-web/guide/introduction.html) (i.e., the copy&paste 
drop-in JavaScript Code) and for [Salesforce Enhance Chat Custom Client](https://developer.salesforce.com/docs/service/messaging-api/guide/get-started.html) (i.e., where you use a 
REST API directly).
